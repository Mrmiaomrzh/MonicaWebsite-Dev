# ⌨️ Monica Android 自动填充方式

::: note 概述
为了防止因系统或应用兼容性问题导致自动填充无法使用，Monica for Android 提供了三种不同的自动填充方式以及一种最终保底机制，确保用户在各种应用场景下都能顺利完成填充。
:::

---

## 🛠️ 三种自动填充方式

### 1. 自动填充 autofill
* **原理**：利用 Android 系统提供的标准 autofill 链路进行自动填充。
* **开启方法**：在系统设置中选择 `Monica autofill` 为自动填充服务。

### 2. Monica 键盘
* **原理**：类似于“密码键盘”应用。需要聚焦表单，然后点击键盘上的填充按钮，通过 <kbd>Tab</kbd> 键切换遍历表单，按照字段进行自动填充。
* **开启方法**：在系统设置的输入法设置中，勾选启用 `Monica 键盘`。

### 3. 控制栏磁贴（无障碍）
* **原理**：利用系统的无障碍权限进行自动填充，作为以上两种方案的保底机制。
* **开启方法**：在下拉控制栏中添加 Monica 磁贴，并在系统设置的无障碍服务中给予 `Monica Pass` 无障碍权限。

---

## 🛡️ 最终保底机制

如果以上三种填充方式皆不可用，可以通过结合**控制栏磁贴**与**通知权限**来实现快速复制填充。

### 开启方法
* 在下拉控制栏添加磁贴。
* 在系统设置中给予 `Monica Pass` **通知权限**。

### 操作步骤
1. 下拉控制栏，点击 **Monica 磁贴** 进入密码自动填充页面。
2. 点击想要填充的密码列表项，选择 **“复制账号稍后复制密码”**。
3. 此时系统会马上复制账号到剪贴板，同时发送密码到系统通知栏。
4. 粘贴完账号后，**点击通知栏的该条通知**，即可自动复制密码，完成后续粘贴。
