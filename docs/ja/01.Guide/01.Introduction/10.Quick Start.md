### 迅速なインストール

Android:
1. GitHub の [Releases](https://github.com/JoyinJoester/Monica/releases) ページから最新の APK ファイルをダウンロードします。
2. Android 8.0 以上のデバイスにインストールし、起動後にマスターパスワードを初期設定します。

ブラウザ拡張機能 (Chrome / Edge):
1. `Monica for Browser` ディレクトリ内で拡張機能をビルドします。
2. ブラウザで `chrome://extensions/` を開き、「デベロッパー モード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」を選択し、ビルドされた `dist` ディレクトリを選択して導入します。

### 既知の制限事項
- システムの互換性の原因により、現在、一部のシャオミ製 HyperOS デバイスにおいて、Monica for Android からパスキー（Passkey）を正常に作成できない問題が確認されています。
