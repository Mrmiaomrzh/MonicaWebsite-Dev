# ⌨️ Monica Android 自動入力方式

::: note 概要
システムやアプリケーションの互換性問題によって自動入力ルーチンが動作しなくなる事態を防ぐため、Monica for Android は 3 つの異なるプライマリ自動入力モードと、最終手段としての保底（セーフガード・フォールバック）メカニズムを提供し、あらゆるアプリ環境下でもユーザーがスムーズに入力を完了できるようにしています。
:::

---

## 🛠️ 3つのプライマリ自動入力モード

### 1. 標準システム自動入力 (Autofill)
* **仕組み**：Android システムが提供する標準の Autofill フレームワークを利用して、パスワードフォームへのマッチングと入力を実行します。
* **有効化方法**：システムの「設定」を開き、デフォルトの自動入力サービスとして `Monica autofill` を選択します。

### 2. Monica カスタムキーボード
* **仕組み**：独立したパスワード管理用ソフト入力法（パスワードキーボード）として機能します。ユーザーがターゲットの入力ボックスにフォーカスを当て、カスタムレイアウト上の入力ボタンをタップすると、<kbd>Tab</kbd> キーの切り替え制御を利用して各フォームを巡回し、順番にデータを流し込みます。
* **有効化方法**：システムの設定から「画面キーボード（入力法）」設定に移動し、`Monica キーボード` の有効化チェックボックスをオンにします。

### 3. クイック設定タイル（アクセシビリティ）
* **仕組み**：Android のアクセシビリティ（ユーザー補助）サービス権限を利用して、アクティブな画面のレイアウトツリーに強制的に文字列を注入します。標準の自動入力リスナーが抑制されているアプリに対する強力なバックアップルートとして機能します。
* **有効化方法**：デバイスの通知シェード（クイック設定パネル）を編集して Monica のタイルを追加し、システム設定の「ユーザー補助」で `Monica Pass` にアクセシビリティ権限を付与します。

---

## 🛡️ 最終保底（フォールバック）メカニズム

深刻な環境制限により上記の 3 つの自動化経路がすべてブロックされた場合、**クイック設定タイル** と **通知エンジン** の連携を利用した、堅牢な手動コピー＆ペーストブリッジを展開できます。

### 構成手順
* クイック設定のカスタムショートカットを、通知領域のドロップダウンパネルに追加します。
* Android のアプリ設定で、`Monica Pass` に対する **通知権限** を完全に許可します。

### 操作手順
1. システムの通知シェードを引き下げ、**Monica タイル** をタップしてパスワード金庫のオーバーレイ画面を呼び出します。
2. 入力したい認証情報項目をタップし、**「アカウントをコピーし、パスワードを通知に保留」** を選択します。
3. ランタイムは即座にアカウント文字列をグローバルクリップボードにコピーすると同時に、システムの通知バーにパスワード転送用の通知をピン留めします。
4. アプリのアカウント欄にペースト（貼り付け）した後、**通知バーを引き下げてピン留めされた Monica の通知ブロックをクリック** すると、リスナーが作動して対応するパスワード文字列がクリップボードに自動コピーされ、残りのフォームへの即時入力が可能になります。
