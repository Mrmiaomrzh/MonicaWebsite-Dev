# ⌨️ Методы автозаполнения Monica Android

::: note Обзор
Чтобы предотвратить сбои в работе подпрограмм автозаполнения из-за специфических особенностей операционной системы или проблем с совместимостью в сторонних приложениях, Monica для Android предоставляет три уникальных метода автоматического ввода учетных данных наряду с финальным отказоустойчивым механизмом ручного заполнения. Эта двухпутевая структура гарантирует, что пользователи смогут чисто и без задержек вводить учетные данные в любых сценариях использования мобильного устройства.
:::

---

## 🛠️ Три основных метода автозаполнения

### 1. Стандартное системное автозаполнение (Standard System Autofill)
* **Базовый принцип**: Использует стандартную архитектуру Android Autofill Framework для сопоставления и предоставления записей паролей на основе структуры фокуса полей ввода в активном приложении.
* **Как включить**: Перейдите в системные настройки Android -> «Язык и ввод» / «Пароли и автозаполнение» и выберите `Monica autofill` в качестве активного глобального поставщика услуг автозаполнения.

### 2. Кастомная клавиатура Monica (Monica Custom Keyboard)
* **Базовый принцип**: Функционирует аналогично специализированным мягким методам ввода (софт-клавиатурам) менеджеров паролей. Пользователь фокусируется на целевом текстовом поле, переключает клавиатуру на раскладку Monica, выбирает нужную запись и использует элемент управления <kbd>Tab</kbd> для последовательного перемещения по полям и автоматической вставки переменных.
* **Как включить**: Перейдите в системные настройки виртуальных клавиатур и установите флажок для активации движка ввода `Monica Keyboard`.

### 3. Плитка быстрых настроек через Специальные возможности (Quick Settings Tile / Accessibility)
* **Базовый принцип**: Использует примитивы системной службы специальных возможностей Android (`Accessibility Service`) для принудительной инъекции строковых данных в активные деревья разметки интерфейса. Этот метод служит вторичным резервным каналом, когда стандартные слушатели Autofill Framework блокируются или подавляются целевым приложением.
* **Как включить**: Отредактируйте панель быстрых настроек (шторку уведомлений) вашего устройства, добавьте ярлык плитки Monica, а затем предоставьте разрешение приложению `Monica Pass` в разделе системных служб специальных возможностей.

## 🛡️ Абсолютный отказоустойчивый резервный механизм

Если все три основных автоматизированных пути заблокированы жесткими ограничениями операционной среды или кастомной разметкой приложения, вы можете развернуть надежный мост ручной инъекции данных, объединяющий **Плитку быстрых настроек** со слушателями **Движка системных уведомлений**.

### Маршруты конфигурации
* Закрепите кастомный ярлык-плитку на аппаратной панели быстрых настроек в шторке уведомлений.
* Предоставьте `Monica Pass` абсолютные **Разрешения на отправку уведомлений** в системных настройках приложений Android.

### Шаги выполнения

1. Потяните вниз системную шторку уведомлений и нажмите на **Плитку Monica**, чтобы инициировать появление всплывающего оверлей-окна со списком ваших паролей.
2. Найдите и нажмите на необходимую вам учетную запись, после чего выберите опцию **«Скопировать имя пользователя и поставить пароль в очередь уведомлений»** (Copy Username & Queue Password to Notification).
3. Среда выполнения немедленно скопирует имя аккаунта в глобальный буфер обмена системы, одновременно закрепив активный блок уведомления в вашей шторке.
4. Вставьте имя пользователя в форму приложения, затем снова **потяните шторку вниз и нажмите на закрепленный блок уведомления Monica**. Это действие активирует слушатель, который мгновенно скопирует соответствующую строку пароля в буфер обмена, позволяя сразу же развернуть ее в оставшемся пустом поле ввода.
