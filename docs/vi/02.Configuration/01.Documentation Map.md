# 🗺️ Bản đồ điều hướng tài liệu Monica

::: note Tóm tắt
Trang này tổ chức tài liệu Monica theo luồng bắt đầu nhanh, vai trò người dùng và tuyến kỹ thuật để bạn tìm đúng tài liệu trong thời gian ngắn nhất.
:::

## 🚀 Lộ trình bắt đầu nhanh

1. **Hiểu Monica**: Đọc [📖 Giới thiệu dự án](/vi/guide/intro) để nắm phạm vi và mục tiêu thiết kế.  
2. **Nắm toàn cảnh**: Xem [🎯 Tổng quan giao diện](/vi/guide/overview) để hiểu chức năng chính và bố cục UI.  
3. **Khởi động nhanh**: Làm theo [📝 Khởi động nhanh](/vi/guide/quickstart) cho lần thiết lập đầu tiên.  
4. **Đi sâu phần cấu hình**: Tiếp tục tại [⚙️ Trang cấu hình](/vi/reference).
