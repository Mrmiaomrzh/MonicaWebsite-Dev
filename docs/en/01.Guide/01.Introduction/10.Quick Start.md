### Quick Installation

Android:
1. Download the latest APK build from the official [Releases](https://github.com/JoyinJoester/Monica/releases) page.
2. Install the package on any device running Android 8.0+ and configure your master password.

Browser Extension (Chrome / Edge):
1. Navigate to the local `Monica for Browser` project directory to build the extension assets.
2. Open `chrome://extensions/` inside your browser and enable "Developer mode."
3. Click "Load unpacked" and select the generated `dist` distribution folder.

### Known Limitations
- Due to system-level configuration variances, Monica for Android is currently unable to initialize or generate Passkeys on certain devices running Xiaomi HyperOS.
